# Молекула бар - Система контроля доступа

Это система контроля доступа для "Молекула бар", созданная с использованием Next.js, Firebase и ShadCN UI.

## Возможности

- **Список гостей**: Просмотр, поиск и фильтрация всех зарегистрированных гостей.
- **Сканер QR-кодов**: Быстрая проверка гостя по его ID для предоставления доступа.
- **Журнал посещений**: Просмотр истории посещений по датам в удобном календаре.
- **Черный список**: Возможность блокировки доступа для нежелательных гостей.

## Начало работы

### Требования

- [Node.js](https://nodejs.org/) (версия 20.x или выше)
- `npm` или `yarn`

### Установка

1. Склонируйте репозиторий (или скачайте файлы проекта).
2. Установите зависимости:
   ```bash
   npm install
   ```

### Запуск в режиме разработки

Чтобы запустить приложение локально, выполните команду:

```bash
npm run dev
```

Откройте [http://localhost:9002](http://localhost:9002) в вашем браузере, чтобы увидеть результат.

### Настройка Firebase

Проект уже настроен для работы с Firebase. Файл конфигурации находится по пути `src/lib/firebase.ts`. Данные хранятся в Firestore, а правила безопасности — в файле `firestore.rules`.
